# Cursova  проєкт

Короткий опис

Це простий інтернет-магазин на Laravel для навчального курсу. Мета  продемонструвати CRUD для товарів і категорій, а також базову автентифікацію.

##  Встановлення

1. Клонувати репозиторій:
   ```bash
   git clone https://github.com/korzach1998-ai/cursova.git
   ```
2. Перейти в каталог:
   ```bash
   cd cursova
   ```
3. Встановити залежності PHP і JS:
   ```bash
   composer install
   npm install
   ```
4. Створити файл конфігурації середовища і згенерувати ключ:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```
5. Налаштувати підключення до бази даних у `.env` та виконати міграції й наповнення (seed):
   ```bash
   php artisan migrate --seed
   ```
6. Зібрати фронтенд-активи:
   ```bash
   npm run dev
   ```

##  Запуск

Запустити локальний сервер:
```bash
php artisan serve
```
Потім відкрити: http://127.0.0.1:8000

##  Тестування

Запустити тести:
```bash
php artisan test
```
